# VideoGata

A plugin based web app that plays videos.

https://www.videogata.com

## Running locally

```console
npm install
npm run dev
```

## Plugins

[youtube-videogata](https://github.com/InfoGata/youtube-videogata)

[peertube-videogata](https://github.com/InfoGata/peertube-videogata)
